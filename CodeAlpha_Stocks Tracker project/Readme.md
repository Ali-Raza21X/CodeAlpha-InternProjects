Stock Portfolio Tracker (Python CLI)

A simple command-line Stock Portfolio Tracker built with Python. Users can select a stock, enter the quantity, and calculate the total investment value.

Features
Displays available stocks
Accepts stock symbol input
Calculates total investment
Handles invalid stock names
Simple portfolio summary
Technologies Used
Python
Dictionaries
User Input
Conditional Statements (if-else)
How to Run
Install Python.
Save the code as stock_tracker.py.
Run the file:
python stock_tracker.py
Example
_________STOCK TRACKER________

Available Stocks: AAPL, TSLA, GOOG

Enter your stock name: AAPL
Enter your amount: 5

_____ PORTFOLIO DETAILS _____

Stock Name: AAPL
Stock Price: 180
Quantity: 5
Total Investment: 900

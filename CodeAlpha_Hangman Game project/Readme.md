Hangman Game (Python CLI)

A simple command-line Hangman game built with Python. The player guesses letters to reveal a hidden word before running out of lives.

Features
Random word selection
Letter-by-letter guessing
Input validation
Tracks already guessed letters
6 lives system
Win/Lose conditions
Technologies Used
Python
Lists
Loops (while, for)
Conditional Statements (if-else)
random module
How to Run
Install Python.
Save the code as hangman.py.
Run the file:
python hangman.py
Game Rules
Guess one letter at a time.
Correct guesses reveal letters in the word.
Wrong guesses reduce your lives.
You have 6 lives to guess the complete word.
Repeated guesses are not allowed.
Example
_ _ _ _ _

Enter the letter: a
Correct
a _ _ _ _

Enter the letter: z
Wrong letter! Lives Left: 5
